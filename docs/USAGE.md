## 安装部署

### 1. 构建脚本

在项目根目录运行：

```bash
node build.js
```

构建成功后，会在 `dist/` 目录下生成 `命定之诗总结脚本.js` 文件。

### 2. 导入到 SillyTavern

1. 打开 SillyTavern
2. 进入脚本管理界面
3. 导入构建生成的 `命定之诗总结脚本.js` 文件
4. 启用脚本

## 基本使用

### 自动总结

脚本会根据配置自动在合适的时机触发总结：

- 当聊天消息达到设定的阈值时
- 在特定的对话节点
- 根据自定义的触发条件

### 手动总结

通过设置面板可以手动触发总结：

1. 打开脚本设置面板
2. 点击"手动总结"按钮
3. 等待总结生成
4. 查看并确认总结内容

## 配置说明

### API 设置

#### 使用 SillyTavern 默认 API

默认情况下，脚本使用 SillyTavern 的内置 API。无需额外配置。

#### 使用自定义 API

如果需要使用自定义 API：

1. 在设置面板中启用"使用自定义 API"
2. 填写 API 端点地址
3. 配置必要的认证信息（如需要）
4. 测试连接

### 世界书设置

脚本会自动将总结内容存入世界书：

- **自动绑定**：脚本会自动检测并绑定相关世界书
- **条目管理**：支持查看、编辑、删除世界书条目
- **分类标签**：可为不同类型的总结添加标签

### 提示词配置

可以自定义总结的提示词模板：

```javascript
// 示例提示词结构
{
  "system": "你是一个专业的对话总结助手...",
  "user": "请总结以下对话内容：\n{messages}",
  "parameters": {
    "temperature": 0.7,
    "max_tokens": 500
  }
}
```

## 功能详解

### 消息提取

脚本会智能提取聊天消息：

- 过滤系统消息
- 识别用户和角色消息
- 保留重要的上下文信息
- 支持自定义过滤规则

### 总结生成

总结生成流程：

1. **提取消息**：从聊天历史中提取相关消息
2. **构建提示**：根据配置构建总结提示词
3. **调用 API**：发送请求到 AI 模型
4. **处理响应**：解析并格式化总结内容
5. **存入世界书**：将总结保存到世界书条目

### 世界书集成

世界书功能：

- **自动创建条目**：为每次总结自动创建世界书条目
- **关键词提取**：自动提取关键词作为触发词
- **优先级管理**：支持设置条目优先级
- **条目编辑**：可在面板中直接编辑条目内容

### 设置面板

设置面板提供以下功能：

- 📊 **状态显示**：查看当前脚本运行状态
- ⚙️ **参数配置**：调整各项配置参数
- 📝 **条目列表**：查看和管理所有总结条目
- 🔄 **重新生成**：对已有总结重新生成
- 🗑️ **删除条目**：删除不需要的总结

## 常见问题

### Q: 脚本无法加载？

**A:** 请检查：
- SillyTavern 版本是否兼容
- 脚本文件是否完整
- 浏览器控制台是否有错误信息

### Q: 总结内容不准确？

**A:** 可以尝试：
- 调整提示词模板
- 修改温度参数（temperature）
- 增加上下文消息数量
- 使用更强大的 AI 模型

### Q: 世界书条目未创建？

**A:** 请确认：
- 世界书是否已正确绑定
- 是否有足够的权限创建条目
- 检查控制台错误日志

### Q: API 调用失败？

**A:** 排查步骤：
- 检查 API 地址是否正确
- 验证认证信息是否有效
- 确认网络连接正常
- 查看 API 配额是否用尽

### Q: 如何自定义总结格式？

**A:** 修改提示词模板中的格式说明，例如：

```
请按以下格式总结：
1. 主要事件：...
2. 角色动态：...
3. 重要信息：...
```
